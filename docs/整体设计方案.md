# 智衡(Balancer) - MVVM架构完整设计方案

经过检查，之前的方案**不完全符合MVVM架构**。Vue组件虽然使用了Composition API，但缺少明确的ViewModel层分离。现在重新设计一套严格的MVVM架构方案。

## 一、MVVM架构设计总览

### 1.1 架构层次分离
```
┌─────────────────────────────────────────────────┐
│                    View层 (UI)                  │
│  纯展示组件，无业务逻辑，只负责渲染和用户交互      │
│  - Vue组件 (Template部分)                       │
│  - 样式文件                                      │
│  - 布局组件                                      │
└─────────────────────────────────────────────────┘
                          │ 双向数据绑定
┌─────────────────────────────────────────────────┐
│               ViewModel层 (逻辑)                │
│  视图逻辑，状态管理，数据转换，事件处理           │
│  - Composition函数                              │
│  - 自定义Hooks                                  │
│  - 服务代理                                      │
└─────────────────────────────────────────────────┘
                          │ 数据访问
┌─────────────────────────────────────────────────┐
│                    Model层 (数据)               │
│  数据模型，业务逻辑，数据访问，持久化             │
│  - 实体类 (TypeScript接口/类)                   │
│  - 数据服务 (Service类)                         │
│  - 存储层 (Tauri Commands + SQLite)            │
└─────────────────────────────────────────────────┘
```

```typescript
export interface Balance {
  id: string
  platform: PlatformType
  currentBalance: number
  currency: Currency
  lastUpdated: Date
  history: BalanceHistory[]
  metadata: Record<string, any>
}

export interface BalanceHistory {
  id: string
  balanceId: string
  timestamp: Date
  balance: number
  change: number
  reason: BalanceChangeReason
  metadata?: Record<string, any>
}

export enum PlatformType {
  DEEPSEEK = 'deepseek',
  OPENAI = 'openai',
  ANTHROPIC = 'anthropic',
  CUSTOM = 'custom'
}

export enum Currency {
  CNY = 'CNY',
  USD = 'USD',
  EUR = 'EUR'
}

export enum BalanceChangeReason {
  API_USAGE = 'api_usage',
  MANUAL_UPDATE = 'manual_update',
  SYSTEM_UPDATE = 'system_update',
  CORRECTION = 'correction'
}

// 实体类实现
export class BalanceEntity implements Balance {
  constructor(
    public id: string,
    public platform: PlatformType,
    public currentBalance: number,
    public currency: Currency,
    public lastUpdated: Date,
    public history: BalanceHistory[] = [],
    public metadata: Record<string, any> = {}
  ) {}

  static create(params: Partial<Balance>): BalanceEntity {
    return new BalanceEntity(
      params.id || crypto.randomUUID(),
      params.platform || PlatformType.DEEPSEEK,
      params.currentBalance || 0,
      params.currency || Currency.CNY,
      params.lastUpdated || new Date(),
      params.history || [],
      params.metadata || {}
    )
  }

  addHistoryEntry(entry: Omit<BalanceHistory, 'id' | 'balanceId'>): void {
    const historyEntry: BalanceHistory = {
      id: crypto.randomUUID(),
      balanceId: this.id,
      ...entry
    }
  
    this.history.unshift(historyEntry)
    this.lastUpdated = new Date()
  }

  getDailyChange(): number {
    const yesterday = new Date(Date.now() - 24 * 60 * 60 * 1000)
    const recentHistory = this.history.filter(h => 
      new Date(h.timestamp) > yesterday
    )
  
    if (recentHistory.length === 0) return 0
  
    const oldest = recentHistory[recentHistory.length - 1]
    return this.currentBalance - oldest.balance
  }
}
```

### 2.2 数据仓库接口
```typescript
// src/models/repositories/BaseRepository.ts
export interface IRepository<T, ID> {
  findById(id: ID): Promise<T | null>
  findAll(filter?: Partial<T>): Promise<T[]>
  create(entity: Omit<T, 'id'>): Promise<T>
  update(id: ID, entity: Partial<T>): Promise<T>
  delete(id: ID): Promise<boolean>
  count(filter?: Partial<T>): Promise<number>
}

export abstract class BaseRepository<T, ID> implements IRepository<T, ID> {
  protected abstract tableName: string
  protected abstract toEntity(row: any): T
  protected abstract fromEntity(entity: T): Record<string, any>

  async findById(id: ID): Promise<T | null> {
    try {
      const result = await this.executeQuery(
        `SELECT * FROM ${this.tableName} WHERE id = ?`,
        [id]
      )
      return result.length > 0 ? this.toEntity(result[0]) : null
    } catch (error) {
      throw new RepositoryError(`Failed to find by id: ${id}`, error)
    }
  }

  async findAll(filter?: Partial<T>): Promise<T[]> {
    try {
      let query = `SELECT * FROM ${this.tableName}`
      const params: any[] = []
    
      if (filter && Object.keys(filter).length > 0) {
        const conditions = Object.keys(filter).map(key => `${key} = ?`)
        query += ` WHERE ${conditions.join(' AND ')}`
        params.push(...Object.values(filter))
      }
    
      const results = await this.executeQuery(query, params)
      return results.map(row => this.toEntity(row))
    } catch (error) {
      throw new RepositoryError('Failed to find all', error)
    }
  }

  async create(entity: Omit<T, 'id'>): Promise<T> {
    try {
      const entityWithId = { id: crypto.randomUUID(), ...entity }
      const data = this.fromEntity(entityWithId as T)
      const columns = Object.keys(data).join(', ')
      const placeholders = Object.keys(data).map(() => '?').join(', ')
    
      await this.executeQuery(
        `INSERT INTO ${this.tableName} (${columns}) VALUES (${placeholders})`,
        Object.values(data)
      )
    
      return entityWithId as T
    } catch (error) {
      throw new RepositoryError('Failed to create entity', error)
    }
  }

  protected abstract executeQuery(sql: string, params?: any[]): Promise<any[]>
}

// 错误类
export class RepositoryError extends Error {
  constructor(
    message: string,
    public readonly originalError?: any
  ) {
    super(message)
    this.name = 'RepositoryError'
  }
}
```

```typescript
// src/models/repositories/BalanceRepository.ts
import { BaseRepository, RepositoryError } from './BaseRepository'
import { Balance, BalanceEntity, BalanceHistory } from '../entities/Balance'
import { invoke } from '@tauri-apps/api/tauri'

export class BalanceRepository extends BaseRepository<Balance, string> {
  protected tableName = 'balances'

  protected toEntity(row: any): Balance {
    return BalanceEntity.create({
      id: row.id,
      platform: row.platform,
      currentBalance: row.current_balance,
      currency: row.currency,
      lastUpdated: new Date(row.last_updated),
      history: JSON.parse(row.history || '[]'),
      metadata: JSON.parse(row.metadata || '{}')
    })
  }

  protected fromEntity(entity: Balance): Record<string, any> {
    return {
      id: entity.id,
      platform: entity.platform,
      current_balance: entity.currentBalance,
      currency: entity.currency,
      last_updated: entity.lastUpdated.toISOString(),
      history: JSON.stringify(entity.history),
      metadata: JSON.stringify(entity.metadata)
    }
  }

  protected async executeQuery(sql: string, params?: any[]): Promise<any[]> {
    try {
      return await invoke<any[]>('execute_sql', { sql, params })
    } catch (error) {
      throw new RepositoryError(`SQL execution failed: ${sql}`, error)
    }
  }

  // 自定义查询方法
  async findByPlatform(platform: string): Promise<Balance[]> {
    return this.findAll({ platform } as Partial<Balance>)
  }

  async updateBalance(id: string, newBalance: number, reason: string): Promise<Balance> {
    const balance = await this.findById(id)
    if (!balance) {
      throw new RepositoryError(`Balance not found: ${id}`)
    }
  
    const change = newBalance - balance.currentBalance
  
    // 添加历史记录
    const historyEntry: Omit<BalanceHistory, 'id' | 'balanceId'> = {
      timestamp: new Date(),
      balance: newBalance,
      change,
      reason: 'api_usage' // 这里应该根据reason映射
    }
  
    const updatedBalance = BalanceEntity.create({
      ...balance,
      currentBalance: newBalance,
      lastUpdated: new Date(),
      history: [historyEntry, ...balance.history]
    })
  
    return await this.update(id, updatedBalance)
  }

  async getBalanceHistory(id: string, days: number = 7): Promise<BalanceHistory[]> {
    try {
      const result = await this.executeQuery(
        `SELECT * FROM balance_history 
         WHERE balance_id = ? 
         AND timestamp >= datetime('now', ?) 
         ORDER BY timestamp DESC`,
        [id, `-${days} days`]
      )
    
      return result.map(row => ({
        id: row.id,
        balanceId: row.balance_id,
        timestamp: new Date(row.timestamp),
        balance: row.balance,
        change: row.change,
        reason: row.reason,
        metadata: JSON.parse(row.metadata || '{}')
      }))
    } catch (error) {
      throw new RepositoryError(`Failed to get balance history for ${id}`, error)
    }
  }
}
```

### 2.4 业务服务层
```typescript
// src/models/services/BalanceService.ts
import { BalanceRepository } from '../repositories/BalanceRepository'
import { Balance, BalanceEntity, PlatformType } from '../entities/Balance'
import { IMonitoringService } from './MonitoringService'

export interface IBalanceService {
  getCurrentBalance(platform: PlatformType): Promise<Balance>
  updateBalance(platform: PlatformType, newBalance: number): Promise<Balance>
  getBalanceTrend(platform: PlatformType, days: number): Promise<BalanceTrend>
  estimateRemainingDays(platform: PlatformType, dailyUsage?: number): Promise<number>
}

export interface BalanceTrend {
  platform: PlatformType
  currentBalance: number
  dailyAverageChange: number
  trend: 'increasing' | 'decreasing' | 'stable'
  history: Array<{
    date: string
    balance: number
    change: number
  }>
}

export class BalanceService implements IBalanceService {
  private balanceRepository: BalanceRepository
  private monitoringService?: IMonitoringService

  constructor(
    balanceRepository?: BalanceRepository,
    monitoringService?: IMonitoringService
  ) {
    this.balanceRepository = balanceRepository || new BalanceRepository()
    this.monitoringService = monitoringService
  }

  async getCurrentBalance(platform: PlatformType): Promise<Balance> {
    try {
      const balances = await this.balanceRepository.findByPlatform(platform)
    
      if (balances.length === 0) {
        // 创建新的余额记录
        const newBalance = BalanceEntity.create({
          platform,
          currentBalance: 0,
          currency: 'CNY',
          lastUpdated: new Date()
        })
      
        return await this.balanceRepository.create(newBalance)
      }
    
      return balances[0]
    } catch (error) {
      throw new ServiceError('Failed to get current balance', error)
    }
  }

  async updateBalance(platform: PlatformType, newBalance: number): Promise<Balance> {
    try {
      const current = await this.getCurrentBalance(platform)
    
      if (Math.abs(current.currentBalance - newBalance) < 0.01) {
        return current // 余额变化太小，不更新
      }
    
      const reason = newBalance < current.currentBalance ? 'api_usage' : 'manual_update'
      const updated = await this.balanceRepository.updateBalance(
        current.id,
        newBalance,
        reason
      )
    
      // 触发余额变更事件
      this.emitBalanceChanged(updated)
    
      // 检查是否需要告警
      await this.checkBalanceAlert(updated)
    
      return updated
    } catch (error) {
      throw new ServiceError('Failed to update balance', error)
    }
  }

  async getBalanceTrend(platform: PlatformType, days: number = 7): Promise<BalanceTrend> {
    try {
      const balance = await this.getCurrentBalance(platform)
      const history = await this.balanceRepository.getBalanceHistory(balance.id, days)
    
      if (history.length < 2) {
        return {
          platform,
          currentBalance: balance.currentBalance,
          dailyAverageChange: 0,
          trend: 'stable',
          history: []
        }
      }
    
      // 计算每日平均变化
      const totalChange = history.reduce((sum, entry) => sum + entry.change, 0)
      const dailyAverageChange = totalChange / days
    
      // 确定趋势
      let trend: 'increasing' | 'decreasing' | 'stable'
      if (dailyAverageChange > 10) {
        trend = 'increasing'
      } else if (dailyAverageChange < -10) {
        trend = 'decreasing'
      } else {
        trend = 'stable'
      }
    
      // 格式化历史数据
      const formattedHistory = history.map(entry => ({
        date: entry.timestamp.toISOString().split('T')[0],
        balance: entry.balance,
        change: entry.change
      }))
    
      return {
        platform,
        currentBalance: balance.currentBalance,
        dailyAverageChange,
        trend,
        history: formattedHistory
      }
    } catch (error) {
      throw new ServiceError('Failed to get balance trend', error)
    }
  }

  async estimateRemainingDays(platform: PlatformType, dailyUsage?: number): Promise<number> {
    try {
      const balance = await this.getCurrentBalance(platform)
    
      if (dailyUsage !== undefined && dailyUsage > 0) {
        return Math.floor(balance.currentBalance / dailyUsage)
      }
    
      // 如果没有提供每日使用量，则基于历史数据估算
      const trend = await this.getBalanceTrend(platform, 30)
      const avgDailyChange = Math.abs(trend.dailyAverageChange)
    
      if (avgDailyChange === 0) {
        return Infinity // 没有使用量，余额不会减少
      }
    
      return Math.floor(balance.currentBalance / avgDailyChange)
    } catch (error) {
      throw new ServiceError('Failed to estimate remaining days', error)
    }
  }

  private async emitBalanceChanged(balance: Balance): Promise<void> {
    // 这里可以触发事件总线或通知服务
    console.log(`Balance changed: ${balance.platform} = ${balance.currentBalance}`)
  
    // 实际项目中可以这样实现：
    // eventBus.emit('balance:changed', balance)
  }

  private async checkBalanceAlert(balance: Balance): Promise<void> {
    // 检查余额是否低于阈值
    const settings = await this.getAlertSettings()
    const threshold = settings.lowBalanceThreshold
  
    if (balance.currentBalance < threshold) {
      // 触发低余额告警
      await this.triggerLowBalanceAlert(balance, threshold)
    }
  }

  private async getAlertSettings(): Promise<{ lowBalanceThreshold: number }> {
    // 从设置服务获取阈值
    return { lowBalanceThreshold: 50 } // 默认50元
  }

  private async triggerLowBalanceAlert(balance: Balance, threshold: number): Promise<void> {
    // 触发告警逻辑
    console.log(`Low balance alert: ${balance.currentBalance} < ${threshold}`)
  
    // 实际项目中可以这样实现：
    // alertService.triggerAlert({
    //   type: 'low_balance',
    //   message: `余额低于阈值: ${balance.currentBalance} < ${threshold}`,
    //   severity: 'warning'
    // })
  }
}
```

```typescript
// src/viewmodels/BaseViewModel.ts
import { ref, reactive, computed, onMounted, onUnmounted } from 'vue'
import type { Ref, Reactive } from 'vue'

export abstract class BaseViewModel<TState = any> {
  protected state: Reactive<TState>
  protected loading: Ref<boolean> = ref(false)
  protected error: Ref<string | null> = ref(null)

  constructor(initialState: TState) {
    this.state = reactive(initialState) as Reactive<TState>
  }

  // 生命周期钩子
  protected async onInitialize(): Promise<void> {}
  protected async onDispose(): Promise<void> {}

  // 公共方法
  public async initialize(): Promise<void> {
    try {
      this.loading.value = true
      this.error.value = null
      await this.onInitialize()
    } catch (err) {
      this.error.value = err instanceof Error ? err.message : 'Unknown error'
      throw err
    } finally {
      this.loading.value = false
    }
  }

  public async dispose(): Promise<void> {
    try {
      await this.onDispose()
    } catch (err) {
      console.error('Error during ViewModel disposal:', err)
    }
  }

  // 状态访问器
  public get isLoading(): boolean {
    return this.loading.value
  }

  public get hasError(): boolean {
    return this.error.value !== null
  }

  public get errorMessage(): string | null {
    return this.error.value
  }

  // 工具方法
  protected async withLoading<T>(operation: () => Promise<T>): Promise<T> {
    this.loading.value = true
    try {
      return await operation()
    } finally {
      this.loading.value = false
    }
  }

  protected setError(message: string): void {
    this.error.value = message
  }

  protected clearError(): void {
    this.error.value = null
  }

  // 响应式访问器
  public get reactiveState(): Reactive<TState> {
    return this.state
  }
}

// Composition函数包装器
export function useViewModel<TViewModel extends BaseViewModel>(
  ViewModelClass: new () => TViewModel,
  ...args: any[]
) {
  const viewModel = new ViewModelClass(...args)

  onMounted(() => {
    viewModel.initialize()
  })

  onUnmounted(() => {
    viewModel.dispose()
  })

  return viewModel
}
```

```typescript
    this.setupAutoRefresh()

    this.setupDataListeners()
  }

  protected async onDispose(): Promise<void> {
    if (this.refreshInterval) {
      clearInterval(this.refreshInterval)
    }
  
    // 清理监听器
    this.cleanupDataListeners()
  }

  // 公共方法 - 供View调用
  public async refresh(): Promise<void> {
    await this.withLoading(async () => {
      await this.loadDashboardData()
    })
  }

  public async updateBalance(platform: PlatformType, newBalance: number): Promise<void> {
    await this.withLoading(async () => {
      await this.balanceService.updateBalance(platform, newBalance)
      await this.loadBalances()
    })
  }

  public dismissAlert(alertId: string): void {
    this.state.recentAlerts = this.state.recentAlerts.filter(
      alert => alert.id !== alertId
    )
  }

  // 计算属性
  public get totalBalance(): number {
    return this.state.balances.reduce(
      (sum, balance) => sum + balance.currentBalance,
      0
    )
  }

  public get hasActiveAlerts(): boolean {
    return this.state.recentAlerts.some(alert => !alert.resolved)
  }

  public get platformCount(): number {
    return this.state.balances.length
  }

      this.loadUsageSummary(),
      this.loadRecentAlerts(),

    ])
  }

    try {
      const platforms: PlatformType[] = [

        PlatformType.OPENAI,
        PlatformType.ANTHROPIC
      ]
    
      const balancePromises = platforms.map(async platform => {
        const balance = await this.balanceService.getCurrentBalance(platform)
        const trend = await this.balanceService.getBalanceTrend(platform)
        const remainingDays = await this.balanceService.estimateRemainingDays(platform)
      
        return {
          platform,
          currentBalance: balance.currentBalance,
          currency: balance.currency,
          trend,
          remainingDays
```

```typescript
import { BaseViewModel } from '../BaseViewModel'
import { BalanceService } from '../../models/services/BalanceService'
import { PlatformType } from '../../models/entities/Balance'

interface DataCardState {
  platform: PlatformType

export class DataCardViewModel extends BaseViewModel<DataCardState> {
  private balanceService: BalanceService
  private autoRefreshInterval?: number

  constructor(platform: PlatformType) {
    const initialState: DataCardState = {
      platform,
      currentBalance: 0,
      currency: 'CNY',
      dailyChange: 0,
      trend: 'stable',
      remainingDays: 0,
      isLoading: false,
      error: null
    }
  
    super(initialState)
  
    this.balanceService = new BalanceService()
  }

  protected async onInitialize(): Promise<void> {
    await this.loadData()
    this.setupAutoRefresh()
  }

  protected async onDispose(): Promise<void> {
    if (this.autoRefreshInterval) {
      clearInterval(this.autoRefreshInterval)
    }
  }

  // 公共方法
  public async refresh(): Promise<void> {
    await this.loadData()
  }

  public async updateBalance(newBalance: number): Promise<void> {
    try {
      this.state.isLoading = true
      await this.balanceService.updateBalance(this.state.platform, newBalance)
      await this.loadData()
    } catch (error) {
      this.state.error = error instanceof Error ? error.message : '更新失败'
      throw error
    } finally {
      this.state.isLoading = false
    }
  }

  // 私有方法
  private async loadData(): Promise<void> {
    try {
      this.state.isLoading = true
      this.state.error = null
    
      const [balance, trend, remainingDays] = await Promise.all([
        this.balanceService.getCurrentBalance(this.state.platform),
        this.balanceService.getBalanceTrend(this.state.platform, 1),
        this.balanceService.estimateRemainingDays(this.state.platform)
      ])
    
      this.state.currentBalance = balance.currentBalance
      this.state.currency = balance.currency
      this.state.dailyChange = trend.dailyAverageChange
      this.state.trend = this.determineTrend(trend.dailyAverageChange)
      this.state.remainingDays = remainingDays
    } catch (error) {
      this.state.error = error instanceof Error ? error.message : '加载失败'
      console.error('Failed to load data card:', error)
    } finally {
      this.state.isLoading = false
    }
  }

  private determineTrend(dailyChange: number): 'up' | 'down' | 'stable' {
    if (dailyChange > 5) return 'up'
    if (dailyChange < -5) return 'down'
    return 'stable'
  }

  private setupAutoRefresh(): void {
    // 每30分钟自动刷新
    this.autoRefreshInterval = setInterval(() => {
      this.loadData()
    }, 30 * 60 * 1000) as unknown as number
  }

  // 计算属性
  public get formattedBalance(): string {
    return `${this.state.currentBalance.toFixed(2)} ${this.state.currency}`
  }

  public get formattedDailyChange(): string {
    const sign = this.state.dailyChange >= 0 ? '+' : ''
    return `${sign}${this.state.dailyChange.toFixed(2)}`
  }

  public get changeColor(): string {
    switch (this.state.trend) {
      case 'up': return 'text-green-500'
      case 'down': return 'text-red-500'
      default: return 'text-gray-500'
    }
  }

  public get progressPercentage(): number {
    const maxDays = 30 // 假设最大显示30天
    return Math.min((this.state.remainingDays / maxDays) * 100, 100)
  }

  public get progressStatus(): 'success' | 'warning' | 'error' {
    if (this.state.remainingDays > 15) return 'success'
    if (this.state.remainingDays > 7) return 'warning'
    return 'error'
  }
}
```

### 5.2 数据流向验证
```
正确的MVVM数据流：
1. View触发事件 → ViewModel处理 → Model执行业务逻辑
2. Model数据变化 → 通知ViewModel → 更新View状态
3. View绑定ViewModel的响应式数据

示例流程：
View (点击刷新按钮) 
  → ViewModel.refresh() 
  → Model.BalanceService.getCurrentBalance()
  → 返回数据到ViewModel
  → ViewModel更新state
  → View自动重新渲染
```

### 5.3 依赖关系验证
```
正确依赖方向：
View → ViewModel → Model
     ↖          ↖
   
错误依赖方向（不允许）：
View → Model ❌
ViewModel → View ❌

本方案确保：
1. View只依赖ViewModel
2. ViewModel只依赖Model
3. Model不依赖任何上层
```

## 七、MVVM架构优势总结

### 7.1 架构优势
1. **关注点分离** - View、ViewModel、Model各司其职
2. **可测试性** - 每层都可以独立测试
3. **可维护性** - 代码结构清晰，易于理解和修改
4. **可复用性** - ViewModel可以在多个View中复用
5. **团队协作** - 前端开发者可以专注于View和ViewModel，后端开发者专注于Model

### 7.2 技术优势
1. **TypeScript全程类型安全** - 从Model到View都有完整类型定义
2. **Vue 3 Composition API** - 完美支持ViewModel模式
3. **Tauri跨平台** - 一套代码，多平台运行
4. **本地存储+GitHub更新** - 满足您的所有需求
5. **现代化工具链** - Vite、Vitest、TypeScript等

### 7.3 业务优势
1. **快速开发** - 清晰的架构加速开发进度
2. **高质量代码** - 分层架构减少bug
3. **易于扩展** - 新增功能只需要在对应层添加代码
4. **长期维护** - 代码结构清晰，新人容易上手
